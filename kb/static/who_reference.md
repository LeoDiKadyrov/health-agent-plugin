# WHO Standard Lab Reference Ranges

Use as baseline comparison. Longevity protocols (Blueprint, Attia) are significantly stricter.
These are population "normal" ranges — not longevity targets.

## Общий анализ крови

| Маркер | Единицы | Норма (муж) | Норма (жен) |
|-|-|-|-|
| Гемоглобин | г/л | 130–170 | 120–155 |
| Эритроциты | ×10¹²/л | 4.5–5.5 | 3.8–5.1 |
| Лейкоциты | ×10⁹/л | 4.0–10.0 | 4.0–10.0 |
| Тромбоциты | ×10⁹/л | 150–400 | 150–400 |
| Гематокрит | % | 40–52 | 36–48 |
