# Sinclair Protocol — Biomarker Targets

Focus: epigenetic reprogramming, information theory of aging. Prioritizes low mTOR signaling, NAD+ restoration, and epigenetic age reduction.

## Ключевые маркеры

| Маркер | Единицы | Цель | Приоритет | Примечание |
|-|-|-|-|-|
| DunedinPACE | темп/год | <0.75 | high | Главный маркер. <0.75 = старение медленнее паспорта |
| Глюкоза натощак | мг/дл | <90 | high | Низкий mTOR сигнал |
| HbA1c | % | <5.3 | high | |
| hs-CRP | мг/л | <1.0 | high | |
| IGF-1 | нг/мл | 100–170 | medium | Нижняя треть нормы для возраста |
| ИМТ | кг/м² | <23 | medium | |
| Инсулин натощак | мкМЕ/мл | <5 | medium | |

## Нутрицевтики (контекст анализа)

Маркеры, на которые влияет стек Sinclair:

| Нутрицевтик | Цель маркер | Ожидаемый эффект |
|-|-|-|
| NMN/NR 1г/день | NAD+ (если измеряется) | ↑ NAD+ |
| Ресвератрол 1г/день | hs-CRP | ↓ воспаление |
| Метформин 1г/день | Глюкоза, HbA1c | ↓ глюкоза |
| Рапамицин (пульс) | IGF-1 | ↓ mTOR |
